# Simon Game 🎮

A fun and interactive Simon Game built using HTML, CSS, and JavaScript. Challenge your memory by repeating random sequences of flashing colors and sounds!

## 🚀 Features

- Classic Simon Game logic
- Responsive design for all screen sizes
- Sound and visual effects
- Incrementally difficult levels
- Lightweight and fast

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
